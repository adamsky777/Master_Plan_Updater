v2.2.0

  • Switch from charger events to completed tasks related to tasks completed.
  • UTF-8 support for Waiting for parts.
  • Rebalance tasks automation.

    • Date: Aug 16th 2023

v2.1.8

  • Changes related to backlog calculation.
  • Waiting for parts is taken from the actual waiting from parts automation tool.
  • Total backlog is aggregated by inventory sites.
  • Vehicle inflow and vehicle outflow: per city per vehicle model.
  • Check whether cities in the city list are in the Revamped dictionary (inventory site mapping).
  • Check whether cities in the city list are updated by the inventory automation tool prior user launches the updating procedure.
  • Display error message to the user if the mentioned two cases are false.

    • Date: June 21st 2023

v2.1.7

  • Backlog Added. Waiting for Service can be calculated -> Backlog - waiting for parts.
  • Checkbox Collection deployment swaps separated for 3PL and in house worker

  • Resolves an issue when user tried to load user data from a different user causing the program to collapse. It notifies the user and prevents loading data from other user. (city list are interchangeable)

    • Date: February 20th 2023
